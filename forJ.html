For

El ciclo For o Para provee una forma concisa de escribir una estructura de repetición. A diferencia del ciclo While los elementos que necesita esta estructura son: una inicialización, una condición y un incremento o decremento. Esta estructura es muy utilizada cuando se conoce el numero de iteraciones que realizará el ciclo. Su sintaxis es la siguiente:

for(inicialización; condición; paso){Codigo}

En la sintaxis anterior la inicialización corresponde a una variable que almacenará el valor inicial de donde empezará el ciclo. La condición es aquella evaluación que le permitirá decir al ciclo si continua o no y el paso es el aumento o decremento que se tenga que realizar.

Ejemplo:

for(int i = 0; i <=10; i=i + 1){
 System.out.println(i)
}
En el ejemplo anterior se está usando un ciclo para imprimir en pantalla un conteo que inicia desde cero hasta 10 incrementando el conteo de uno en uno
